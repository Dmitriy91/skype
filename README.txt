В одном солюшене(Skype) 4 проекта:
-Одноименный проект Skype - клиентская сторона,
-Server - консольный сервер,
-MicroMvvm - заготовка для MVVM паттерна(Не мой проект. Его dll используется на клиене),
-Packages - содержит разные пакеты, которые используются для передачи данных.

Для запуска:
-Запустить проект(Server) и подключить базу данных.
-Создать модель DB, Используюя LINQ to SQL дизайнер(открыть .dbml файл в папке DAO). 
DB лежит в директории - Skype/Server/DAO/SkypeDatabase.mdf). 
-Перетащить таблицы в дизайнер. Перетащить хранимые процедуры.
-Запустить клиент(Skype project). Ввести валидные логин(air12) и пароль(hey123123)
